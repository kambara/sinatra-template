# My Sinatra Template

Based on Sinatra, Haml and CoffeeScript.

## Install

1: Install RubyGems

2: Install Bundler (>= 1.0)

    $ sudo gem install bundler

3: Bundle

    $ cd MY/APP
    $ bundle install --path vendor/bundle

4: Install CoffeeScript


### Update Bundle

    $ bundle update

## Usage

Webrick:

    $ ruby main.rb

Rackup:

    $ rackup

Thin:

    $ thin start
